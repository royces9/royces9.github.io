Drain

Summary

A music video for the rhythm game, osu!. You can take a look at the finished product to get an idea of what a storyboard is: Drain and Tegami. The goal is for there also to be a game to play, with the video playing in the background. If you want a better idea of osu! itself, there should be plenty of videos on YouTube.
But before I get to the project you can read a more detailed write-up of the project as a whole from the project leader here. I'll only talk a little bit of the relevant work I did, since some of these projects had some random side things, and other people were working on other parts of the project.
Max (the author of the more detailed write-up) is also the author of the C++ storyboard library that all these projects end up using. It was originally written for use with MS Visual C++, and I used Windows for a little bit to work on the project for Tegami. But when working on Drain, my copy of Windows 10 didn't want to be stable (it keeps blue screening!!!), so I got the library working on Linux with make. These projects are the first time that I collaboratively work on a project with someone as well, so in addition to learning C++, I'm learning the process of working with other people (a first!)
